## Built With

- Ionic 3.9.2
- SaSS 
- Oracle Apex

### Prerequisites

Node JS v.9
Java 
Android Studio

### Install
Install Ionic 3.9.2
```
npm install -g ionic@3.9.2
```
Clone this repo

Install all npm dependencies
```
npm install
```
### Deploy

To deploy the application please write the localserver
```
ionic s
```
It wil show in your browser
